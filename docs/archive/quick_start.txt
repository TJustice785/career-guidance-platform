### Step 1: Create Project Folder (2 minutes)

```bash

mkdir career-guidance-platform
```

### Step 2: Copy All Files (5 minutes)

### Step 3: Firebase Setup (10 minutes)

1. **Create Firebase Project**
   - Go to https://console.firebase.google.com/
   - Create project: `career-guidance-platform`

2. **Enable Services**
   - Authentication â†’ Enable Email/Password

### Step 4: Install Dependencies (5 minutes)

```bash

```

### Step 5: Configure Environment (5 minutes)

**server/.env:**
```env
PORT=5000
NODE_ENV=development
FIREBASE_PROJECT_ID=your-project-id
FIREBASE_PRIVATE_KEY="your-private-key"
FIREBASE_CLIENT_EMAIL=your-email
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-gmail
EMAIL_PASSWORD=your-app-password
CLIENT_URL=http://localhost:3000
```

**client/.env:**
```env
REACT_APP_API_URL=http://localhost:5000/api

REACT_APP_FIREBASE_AUTH_DOMAIN=your-domain
REACT_APP_FIREBASE_PROJECT_ID=your-id
REACT_APP_FIREBASE_STORAGE_BUCKET=your-bucket
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your-sender-id
REACT_APP_FIREBASE_APP_ID=your-app-id
```

### Step 6: Run the Application (3 minutes)

## ğŸ†˜ Need Help?

### Common Issues & Solutions

**Backend won't start:**
```bash
# Check if port 5000 is in use
lsof -i :5000
# Kill process if needed
kill -9 <PID>
```

**Frontend build errors:**
```bash
# Clear cache and reinstall
rm -rf node_modules package-lock.json
npm install
```

**Firebase errors:**
- Double-check credentials in .env files
- Ensure Firebase services are enabled
- Check Firebase rules are applied

**Can't login:**
- Verify email first
- Check user exists in both Auth and Firestore
- Check browser console for errors
