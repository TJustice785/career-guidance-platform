## Troubleshooting Common Issues

### Issue: Backend won't start
**Solution:**
- Check if port 5000 is available
- Verify all environment variables are set
- Check Firebase credentials format
- Look at error logs

### Issue: Frontend won't connect to backend
**Solution:**
- Verify REACT_APP_API_URL is correct
- Check backend is running
- Check CORS settings
- Verify network connectivity

### Issue: Authentication errors
**Solution:**
- Verify Firebase config is correct
- Check email verification
- Ensure user exists in both Auth and Firestore
- Check user role permissions

### Issue: File upload fails
**Solution:**
- Check Firebase Storage rules
- Verify file size is under limit
- Check file type is allowed
- Ensure storage bucket is configured

### Issue: Emails not sending
**Solution:**
- Use Gmail App Password (not regular password)
- Check email credentials
- Verify SMTP settings
- Check spam folder

## Tips for Success

1. **Work Incrementally**: Build and test one feature at a time
2. **Use Git**: Commit frequently with clear messages
3. **Test Often**: Test after implementing each feature
4. **Read Error Messages**: They usually tell you what's wrong
5. **Use Console**: console.log() is your friend for debugging
6. **Ask for Help**: Don't hesitate to ask instructors or classmates
7. **Document**: Write notes as you build
8. **Stay Organized**: Follow the project structure strictly
9. **Time Management**: Start early, don't wait until the last day
10. **Backup**: Push to GitHub regularly
