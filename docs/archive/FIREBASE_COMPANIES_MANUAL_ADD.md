## üìã FIELD-BY-FIELD GUIDE

If you prefer to add fields manually instead of copying JSON:

### Basic Information:
- **name**: (string) Company name
- **industry**: (string) Industry sector
- **location**: (string) City, Country
- **description**: (string) Company overview
- **email**: (string) Contact email
- **website**: (string) Company website
- **phone**: (string) Contact number

### Additional Details:
- **employees**: (number) Number of employees
- **founded**: (number) Year founded
- **logo**: (string) Logo URL

### Arrays:
- **benefits**: (array) List of employee benefits
- **departments**: (array) List of departments

### Timestamps:
- **createdAt**: (timestamp) Current date/time
- **updatedAt**: (timestamp) Current date/time

## üéØ Quick Add Steps

1. **Open Firebase Console**
   - Go to: https://console.firebase.google.com
   - Select your project
   - Click "Firestore Database"

2. **Create/Open Collection**
   - If "companies" collection doesn't exist, click "Start collection"
   - Enter collection ID: `companies`
   - If it exists, click on "companies"

3. **Add Document**
   - Click "Add document"
   - Set Document ID to Auto-ID
   - Add fields one by one OR use JSON import

4. **Copy & Paste**
   - Copy the JSON for each company above
   - Use Firebase's JSON import feature
   - Or add each field manually

5. **Save**
   - Click "Save"
   - Repeat for all 10 companies
